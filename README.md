# 🛒 QuickMart - E-Commerce Website

🚀 **QuickMart** is a modern and responsive **E-Commerce Web Application** inspired by platforms like Amazon. It provides users with a seamless online shopping experience, including browsing products, adding to cart, and managing orders.

---

## 📌 Features

### ✨ User Features:

- 🛍️ Browse products by categories
- 🔍 Search and filter products
- 🛒 Add to cart & remove from cart
- ❤️ Wishlist functionality
- 👤 User authentication (Login/Register)
- 📦 Order placement & tracking

### ✨ Admin Features:

- ➕ Add / Update / Delete products
- 📊 Manage orders
- 👥 Manage users

---

## 🛠️ Tech Stack

**Frontend:**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository:

```bash
git clone https://github.com/your-username/quickmart.git
```

### 2️⃣ Navigate to project folder:

```bash
cd quickmart
```

### 3️⃣ Install dependencies:

```bash
npm install
```

### 4️⃣ Run the project:

```bash
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the root and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
